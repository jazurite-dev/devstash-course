# Adding Items To Collections

Now we want to be able to add items to collections. When we create or edit items, we should be able to select a collection or collections.

Let's run the following:

```text
/feature load Add functionality to add an item to a single or multiple collections.

Add an input to the new/edit item forms where we can select from the available collections to add the item to.

Don't worry about displaying the collection pages yet
```

Run `/feature start`.

Check the add and edit item modals and make sure that you can select multiple collections.

If not, then iterate.

Once the feature is implemented:

```text
/feature test
/feature review
/feature complete
/clear
```
